# Docker Nginx App

## 📌 Description
A containerized Nginx web app serving static content with Docker deployment and port exposure.

<br/>

## 🚀 Features
- Uses official Nginx image
- Custom HTML page
- Port mapping to access app locally

<br/>

## 🛠️ Requirements
- Docker installed

<br/>

## ▶️ Build Image

```bash
docker build -t my-nginx-app .
```
