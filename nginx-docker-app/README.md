# Docker Nginx App

## 📌 Description
A simple Docker project that serves a static HTML page using Nginx.

<br/>

## 🚀 Features
- Uses official Nginx image
- Custom HTML page
- Port mapping to access app locally

<br/>

## 🛠️ Requirements
- Docker installed

<br/>

## ▶️ Build Image

```bash
docker build -t my-nginx-app .
```
